# CSS

### float와 overflow

#### float

요소를 왼쪽 또는 오른쪽에 배치할 때 유용한 속성  
float 속성을 부여한 요소의 다음 요소는 float 속성을 부여한 요소가 차지하는 위치만 제외하고 나머지 공간을 채운다.  
사이트 레이아웃을 만들 때 사용

#### overflow

내용이 요소의 크기를 벗어났을 때 어떻게 처리할지 정하는 속성  
내용이 주어진 공간에 다 들어가지 않을 때, 보여주게 할 수도 있고, 안보이게 할 수도 있고, 스크롤바를 만들게 할 수도 있다.
